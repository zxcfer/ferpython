### Gmail

| Shortcut      |                       |
|---------------|:----------------------|
| `Shift` + `?` | Enable                |
| `Ctrl` + `b`  | Bold                  |
| `Ctrl` + `i`  | Italic                |
| `Ctrl` + `u`  | Underline             |
| `c`           | compose               |
| `Ctrl` + `\`  | Remove formatting     |
| `=`           | mark as important     |
| `-`           | mark as not important |
